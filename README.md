## What is HugOps?
> Put simply, HugOps is a way to celebrate the human, empathetic side of building, shipping, and running software. It is a take on the word ‚ÄúDevOps‚Äù (breaking down silos between Dev and Ops teams through communication, empathy, etc. to ultimately ship faster), but it extends to folks on all different Software, Support, and IT teams.
